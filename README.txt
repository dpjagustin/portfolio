Mi portfolio - Agustín Perez

Este proyecto se trata de mi portfolio, con el objetivo de dar visibilidad a mi perfil y a mis proyectos con una finalidad laboral o colaborativa para algún proyecto de interés.

Se trabajó en el portfolio estático de manera Mobile First realizando el diseño con "Balsamiq Mockups 3" y la maquetación con HTML y CSS (Utilizando el preprocesador Sass).

Paleta de colores (notación hexadecimal):
- Color principal: #14213d - $priColor
- Color secundario: #fca311 - $secColor
- Color terciario: #0bebaf - $terColor
